Rating

Long story short, we use the Elo rating system, which is a very commonly used rating system to determine the relative skill of different players.

Specific settings

Players start with a base rating of 1500. We are not using a rookie status at this time. k-value (the maximum amount of points you can win in a given game is set to 32.
